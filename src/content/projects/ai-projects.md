### Objective
Streamline data-heavy administrative tasks and technical documentation workflows using modern AI architectures.

### Implementation
- Developed Agentic AI (A2A) workflows to automate multi-step technical analysis.
- Implemented Retrieval-Augmented Generation (RAG) for localized knowledge retrieval from technical manuals and compliance documents.
- Utilized Python and CLI tools for Robotic Process Automation (RPA) of legacy ERP data entry.

### Results
- Reduced manual data processing time for compliance reporting.
- Improved accuracy and accessibility of internal technical documentation.
